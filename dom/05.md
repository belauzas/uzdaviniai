Surasti h1 (antraste)

Perrasyti h1 elemento teksta i "Zuikio pomidoru darzas"

Padaryk, jog h1 elementas turetu CSS class "pavadinimas"

Surasti visus ingredientus

Kiekvieno ingrediento teksta pakeisti i "Pomidoras"

Kiekvienam ingredientui suteikti CSS class "pomidoras"

```html
<h1>Sriubos receptas</h1>
<ul>
  <li>Morkos</li>
  <li>Bulves</li>
  <li>Svogunas</li>
  <li>Vanduo</li>
  <li>Druska</li>
</ul>
```

```css
.pavadinimas {
  text-transform: uppercase;
  font-style: italic;
  font-size: 40px;
}
.pomidoras {
  color: red;
  font-size: 20px;
}
```
